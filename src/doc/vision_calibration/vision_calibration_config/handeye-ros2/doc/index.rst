*******
handeye
*******

HandEye is a ROS package for solving the HandEye camera calibration problem in
the form of :math:`AX = XB`.

- Website: http://wiki.ros.org/handeye
- Source: https://github.com/crigroup/handeye
- Bug reports: https://github.com/crigroup/handeye/issues
